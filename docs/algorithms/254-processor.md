# ABT-2.5.4-Processor: 동작기반 작동 v0.10

### 🔗 References
- KWCAG 2.2: 2.5.4 동작기반 작동
- WCAG 2.2: 2.5.4 Motion Actuation (A) (Reference)

---

이 알고리즘은 **KWCAG 2.5.4(동작기반 작동)** 지침을 준수하며, 기기 흔들기나 기울이기 등 동작 센서를 이용한 기능이 일반적인 인터랙션(클릭, 탭 등)으로도 수행 가능한지 진단한다.

## 1. 데이터 수집 단계 (Data Collection)
- **Script Analysis:** `devicemotion`, `deviceorientation` 관련 이벤트 리스너 사용 여부.
- **Sensor APIs:** 가속도계, 자이로스코프 API 호출 흔적 탐색.

## 2. 분석 및 분류 로직 (Analysis Pipeline)

### [단계 A] 동작 기반 인터랙션 탐지
- **Rule 1.1:** 스크립트 내에서 기기 동작 관련 키워드가 감지되는가?
  - 결과: 감지 시 **[검토 필요]**.
  - 가이드: "기기 동작 센서를 활용한 기능이 탐지되었습니다. 동작뿐만 아니라 버튼 클릭 등 정적인 방법으로도 동일한 기능을 수행할 수 있는지, 그리고 동작 기능을 비활성화할 수 있는지 확인하세요."

## 3. 최종 상태 정의 (Final Status)
1. **검토 필요:** 동작 기반 기능 사용이 의심됨.
2. **적절:** 특수 동작 없이 모든 기능을 조작 가능하거나, 정적 대체 수단이 명확히 제공됨.
3. **N/A:** 기기 센서를 사용하는 코드가 전혀 발견되지 않음.
